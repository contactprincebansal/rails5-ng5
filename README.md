# README

Starter application angular 5 with rails 5 API.

## Requirments

Angular CLI: 1.6.8
Node: 9.0.0
Rails 5.1.5
Ruby 2.4.1

## Local Development

* Run bin/ng-local in terminal 1.
* Run rails s in terminal 2.

## Deployment

* Create a heroku account.
* Add remote to git.
* Run git push heroku master
